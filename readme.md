## About SMA Monitor

SMA Monitor is a web application to monitor the SMA inverter device.

## Deployment

Run migration

```
php artisan migrate
```

Create pasword grant client
```
php artisan passport:client --password
```

Generate passport keys
```
php artisan passport:keys
```
